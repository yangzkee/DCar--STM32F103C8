# Mac Makefile Entry

The buildable Makefile project is `DFCom_Example`.

From this `client_examples` folder:

```sh
make
make flash
```

Or enter the project folder:

```sh
cd DFCom_Example
make
make flash
```

See `DFCom_Example/README_Mac_Makefile.md` for tool installation, wiring, flashing, and MCU notes.
